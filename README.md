## Key Features

* Multiple opened files in tabs.
* Tail file changes.
* Filter log levels.
* Colored log entries.
* Customize log levels pattern by file.
* Customize log levels color by file.
* Cross platform
  - Windows, macOS and Linux ready.

## Installation

```bash
# Clone this repository
$ git clone https://github.com/tmoreno/open-log-viewer

# Go into the repository
$ cd open-log-viewer

# Install dependencies
$ npm install

# Build app
$ npm run build

# Run the app
$ npm run start
```
